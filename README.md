# GlowSync

GlowSync is a Raspberry Pi 5-optimized analytics and monitoring platform for holiday and event light shows.  
It combines:

- Multi-camera RTSP vehicle counting  
- Baldrick Signal phone-presence integration  
- Falcon controller & FPP monitoring  
- Show-hour scheduling, seasons, and exports  
- Web dashboard built with FastAPI + Chart.js

**Default login:** `admin / changeme`  
Edit `.env` to configure credentials and API URLs.
